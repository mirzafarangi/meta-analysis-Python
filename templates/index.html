Why Meta-Mar?

Meta-Mar is a free online meta-analysis service which is developed for department of psychology in university of Marburg, makes it possible to enter data directly or easily upload an already excisting Microsoft Excel spreadsheet in order to receive the results of the analysis.

Facilities of this primary phase of the project include:

  • Possibillity of choosing the Data entry methode between manual Data entry or Excel file uploading.
  • Calculate the effect sizes based on SMD, correlation and risk ration models for each single study.
  • Calculate the overall effect size of the analysis based on fixed and random effect model (include p values, z scores, CI).
  • Calculate the heterogeneity of the analysis (Q Cochran and I2).
  • Run a meta regression (one variable moderator analysis).
  • Demonstrating the Forest Plot of the analysis.
  • Prepare and download the results of the analysis via a .xlsx file.
  • Also availablity of a quick single effect size calculator
